Código refactorizado

En este código podemos observar que tenemos dos arrays de enteros y tenemos que que ordenarlos de menor a mayor. Claramente para ordenar cada uno de ellos tenemos que hacer dos bucles iguales para cada uno de ellos, lo que provoca que tengamos código repetido. Es aquí donde entra en juego la técnica de Extract Method. Este método consiste en recoger ese código que está repetido y crear un método donde alojar ese código. De esta manera, al pasarle por parámetro en este caso el array, solo tendremos que hacer desde el main la llamada al método. Gracias a esta técnica ahorraremos tiempo y conseguiremos tener un código mucho más limpio y óptimo.
