En este caso podemos observar que tenemos una clase en la que guardamos información de dos objetos que pueden ser diferentes como por ejemplo el nombre y apellido no tiene por qué ir vinculado a una cuenta, sino que el nombre y apellido es de una persona. Esos atributos se pueden separar para aliviar de atributos a la clase. Usando el método Extract Class creamos una clase y migramos los atributos que estan relacionados con una persona. Gracias a esta técnica tenedremos un código mucho más limpio, y también nos permite usar esos atributos por separado de una manera más lógica.
