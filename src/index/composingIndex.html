Código refactorizado

En este código, podemos observar que tenemos dos arrays de enteros y necesitamos ordenarlos de menor a mayor. Claramente, para ordenar cada uno de ellos, debemos utilizar dos bucles idénticos, lo que provoca código repetido. Aquí es donde entra en juego la técnica de "Extract Method". Esta técnica consiste en recoger el código repetido y crear un método para alojarlo. De esta manera, al pasarle como parámetro, en este caso, el array, solo necesitaremos llamar al método desde el main. Gracias a esta técnica, ahorraremos tiempo y lograremos tener un código mucho más limpio y eficiente.
